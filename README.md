# GN-AKARI

> Advanced Key Art Rendering Interface

GN-AKARI は、キャラクター画像・テキスト・画像素材を重ねて、  
シーン画像を作成するためのシンプルなローカル Web ツールです。

PSD から立ち絵を生成し、そのまま配置・合成まで行えます。  
素材をレイヤとして配置し、直感的に画像を組み立てる用途を想定しています。

---

## 主な機能

- PSD からキャラクター画像を生成
- 写真や画像をベースにしたシーン合成
- キャラクター・テキスト・画像素材の配置
- レイヤ順の変更（ドラッグ）
- 表示ON/OFF、位置ロック
- メッセージ帯の追加
- 画像素材の管理（設定ページ）

---

## 使い方（最短）

```bash
git clone https://github.com/jin820-dev/GN-AKARI.git
cd GN-AKARI
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

プロジェクトルートで以下を実行してください:

```bash
python src/app.py
```

ブラウザで開く:

http://<サーバのIPアドレス>:5014/
（例: http://192.168.1.10:5014/）

同一 LAN 内の他端末からアクセス可能です。

通常はローカル端末からの利用を推奨します。  
LAN 内の他端末からアクセス可能な設定で起動している場合は、同一ネットワーク上の他者から閲覧される可能性があります。

### 起動オプション

環境変数により起動設定を変更できます。

```bash
AKARI_HOST=0.0.0.0 AKARI_DEBUG=1 python src/app.py
```
- AKARI_HOST: バインドするアドレス（デフォルト: 0.0.0.0）
- AKARI_DEBUG: デバッグモード（1で有効）

## PWA について

GN-AKARI は first release 向けの最小 PWA 対応として、manifest と service worker を含みます。

- `localhost` / `127.0.0.1` ではブラウザからインストールできます
- LAN 公開など `localhost` 以外で利用する場合、ブラウザの service worker / PWA 要件により HTTPS が必要になる場合があります
- オフライン時は最低限のアプリシェルを表示するため、動的 API レスポンスや生成画像は積極的にはキャッシュしません

## データについて

このツールはローカル利用を前提としています。

以下の内容はすべてローカルに保存され、Git 管理対象外です。

- PSD ファイル
- フォント
- 画像素材
- 設定データ
- 出力画像

データは主に以下のディレクトリに保存されます。

- data/
- outputs/
- cache/

### アップロードと削除

アップロード上限は以下です。

- PSD: 50MB
- 画像: 20MB
- フォント: 10MB

削除したファイルは即時削除せず、`data/trash/` に移動します。削除記録は `data/trash/trash_index.json` に残ります。

trash は自動削除しません。必要に応じて手動で復旧・整理してください。

## セキュリティについて

GN-AKARI は、ローカルネットワーク内での個人利用を前提としたツールです。  
認証・権限制御を前提とした設計にはしていません。

デフォルト設定では同一 LAN 内の他端末からアクセス可能です。  
そのため、信頼できるネットワーク環境でのみ使用してください。

また、`debug=True` は開発用の設定です。  
デバッグ情報の表示や開発サーバ向けの挙動を含むため、常用環境では無効化することを推奨します。

推奨:

- 信頼できるローカルネットワーク内で利用する
- 公開サーバ用途には使用しない
- デバッグモードは必要な場合のみ有効化する
