# GN-AKARI

GN-AKARI は、キャラクター画像・テキスト・画像素材を重ねて、
シーン画像を作成するためのシンプルなローカル Web ツールです。

PSD から立ち絵を生成し、そのまま配置・合成まで行えます。

素材をレイヤとして配置し、直感的に画像を組み立てる用途を想定しています。

---

## 主な機能

- PSD からキャラクター画像を生成
- 写真や画像をベースにしたシーン合成
- キャラクター・テキスト・画像素材の配置
- レイヤ順の変更（ドラッグ）
- 表示ON/OFF、位置ロック
- メッセージ帯の追加
- 画像素材の管理（設定ページ）

---

## 使い方（最短）

```bash
git clone https://github.com/<your-name>/GN-AKARI.git
cd GN-AKARI
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

```bash
PYTHONPATH=src flask --app app run --debug --port 5014
```

ブラウザで開く:

```
http://127.0.0.1:5014/
```

※ LAN 内の他の端末からもアクセス可能です

---

## 画面構成

- 立ち絵合成  
  PSD からキャラクター画像を作成

- 写真合成  
  画像・キャラクター・テキストを配置してシーンを作成

- 設定  
  素材やフォントの管理

---

## データについて

このツールはローカル利用を前提としています。

以下の内容はすべてローカルに保存され、Git 管理対象外です。

- PSD ファイル
- フォント
- 画像素材
- 設定データ
- 出力画像

データは主に以下のディレクトリに保存されます。

- data/
- outputs/
- cache/

---

## 注意事項

- 個人利用を想定したツールです
- PSD の再現性は完全ではありません
- レイヤ構成は固定です（追加は未対応）
- 画像オーバーレイは単一レイヤです
