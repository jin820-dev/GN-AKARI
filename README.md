# GN-AKARI

> Advanced Key Art Rendering Interface

GN-AKARI は、キャラクター画像・テキスト・画像素材を重ねて、  
シーン画像を作成するためのシンプルなローカル Web ツールです。

PSD から立ち絵を生成し、そのまま配置・合成まで行えます。  
素材をレイヤとして配置し、直感的に画像を組み立てる用途を想定しています。

v0.2.0 では **素材管理機能（Gallery）を中心に再設計** し、  
操作性・パフォーマンス・スマホ対応を大きく改善しています。

---

## 主な機能

- PSD からキャラクター画像を生成
- 写真や画像をベースにしたシーン合成
- キャラクター・テキスト・画像素材の配置
- レイヤ順の変更（ドラッグ）
- 表示ON/OFF、位置ロック
- メッセージ帯の追加

### Gallery（素材管理）

- 背景 / オーバーレイ素材の一覧管理
- 複数ファイル一括アップロード対応
- 複数選択による一括削除
- サムネイル表示による軽量化
- スマホ最適化UI（stickyタブ・カード最適化）
- 画像プレビュー（モーダル表示）

## 使い方（最短）

```bash
git clone https://github.com/jin820-dev/GN-AKARI.git
cd GN-AKARI
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

プロジェクトルートで以下を実行してください:

```bash
python src/app.py
```

ブラウザで開く:

```
http://<サーバのIPアドレス>:5014/
```

例:

```
http://192.168.1.10:5014/
```

同一 LAN 内の他端末からアクセス可能です。

通常はローカル端末からの利用を推奨します。  
LAN 内公開時は、同一ネットワーク上の他者から閲覧される可能性があります。

---

## 起動オプション

```bash
AKARI_HOST=0.0.0.0 AKARI_DEBUG=1 python src/app.py
```

- AKARI_HOST: バインドするアドレス（デフォルト: 0.0.0.0）
- AKARI_DEBUG: デバッグモード（1で有効）

## 画面構成

- 立ち絵合成  
  PSD からキャラクター画像を作成

- 写真合成（Scene）  
  画像・キャラクター・テキストを配置してシーンを作成

- ギャラリー（Gallery）  
  背景・オーバーレイ素材の管理（追加・削除・一覧表示）

- 設定  
  システム設定（素材管理機能はギャラリーへ集約）

## データについて

このツールはローカル利用を前提としています。

以下の内容はすべてローカルに保存され、Git 管理対象外です。

- PSD ファイル
- フォント
- 画像素材（背景・オーバーレイ）
- 設定データ
- 出力画像

主な保存先:

- data/
  - assets/（背景・オーバーレイ素材）
  - trash/（削除データ）
- outputs/
- cache/

### アップロードと削除

アップロード上限:

- PSD: 50MB
- 画像: 20MB
- フォント: 10MB

削除したファイルは即時削除せず、`data/trash/` に移動されます。  
削除履歴は `data/trash/trash_index.json` に記録されます。

## パフォーマンスについて

- Gallery ではサムネイル表示を採用し、大量画像でも軽快に動作します
- lazy loading に対応し、必要な画像のみ読み込みます
- Scene 側では直近素材のみ表示し、操作性を最適化しています

---

## PWA について

GN-AKARI は最小構成の PWA に対応しています。

- `localhost` / `127.0.0.1` ではインストール可能
- LAN 公開では HTTPS が必要な場合があります
- オフライン時はアプリシェルのみ表示されます

---

## セキュリティについて

GN-AKARI はローカルネットワーク内での個人利用を前提としています。

- 認証・権限制御は実装していません
- 信頼できるネットワーク内で使用してください
- 公開サーバ用途には適していません

推奨:

- ローカルまたは閉じたネットワークで使用
- デバッグモードは必要時のみ有効化

## 注意事項

- 個人利用を想定したツールです
- PSD の再現性は完全ではありません
- レイヤ構成は固定です（追加は未対応）
- 画像オーバーレイは単一レイヤです
- 大量素材利用時は回線環境に依存します（サムネイルで軽量化済み）
